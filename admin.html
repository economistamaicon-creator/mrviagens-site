Configurar GitHub

Configure uma vez e as promoções ficam salvas permanentemente no GitHub.

1

Criar repositório

Acesse github.com/new → Nome: mrviagens-dados → marque Public → desça a página e marque "Add a README file" (obrigatório!) → clique Create repository.

2

Criar Token de acesso

Acesse github.com/settings/tokens/new → Note: mrviagens → Expiration: No expiration → marque repo → clique Generate token → copie o token (começa com ghp_).

3

Preencher abaixo

Cole seu usuário GitHub, nome do repositório e o token gerado:
